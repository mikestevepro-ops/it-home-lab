# Security Labs

Labs covering security fundamentals for CompTIA A+ (Core 2) and TryHackMe.

## Topics Covered

- Malware types and removal (virus, ransomware, spyware, trojans)
- Windows Defender and firewall configuration
- User account security and permissions
- Social engineering awareness
- Basic encryption concepts (BitLocker, HTTPS)
- Physical security concepts
- TryHackMe guided rooms

## Tools Used

- **TryHackMe** — browser-based guided security labs
- **VirtualBox** — safe sandboxed environment
- Windows Security Center / Defender
- Windows Firewall
