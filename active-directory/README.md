# Active Directory Labs

Hands-on Active Directory labs — the bridge between IT help desk and IAM analyst work.

## Why This Folder Matters

Active Directory is the #1 skill used in IT help desk roles (password resets, account lockouts, group membership) and is the direct foundation of IAM analyst work. Every lab here builds toward both.

## Topics Covered

- Setting up a Windows Server 2019 domain controller
- Creating and managing user accounts
- Organizing users with Organizational Units (OUs)
- Creating and applying Group Policy Objects (GPOs)
- Managing security groups and permissions
- Password policies and account lockout policies
- Joining Windows 10 client machines to the domain
- Common help desk tasks: password reset, account unlock, account disable

## Tools Used

- **VirtualBox** — Windows Server 2019 VM + Windows 10 client VM
- Active Directory Users and Computers (ADUC)
- Group Policy Management Console (GPMC)
- PowerShell (basic AD commands)
