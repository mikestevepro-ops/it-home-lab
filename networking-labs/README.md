# Networking Labs

Labs covering networking fundamentals for CompTIA A+ (Core 1) and beyond.

## Topics Covered

- TCP/IP addressing and subnetting
- DNS and DHCP configuration
- Network protocols (HTTP, FTP, SMTP, etc.)
- Network troubleshooting (ping, tracert, ipconfig, nslookup)
- Wireless networking concepts
- Network hardware (switches, routers, hubs)

## Tools Used

- **Cisco Packet Tracer** — network topology simulation
- **VirtualBox** — VM networking configurations
- Windows command line tools
