# IT Home Lab

> Hands-on documentation of my IT home lab — built to prepare for CompTIA A+ certification and develop real-world skills for a career in IT support, IAM, and cloud security.

## About

I'm currently studying for CompTIA A+ (Core 1: 220-1101 & Core 2: 220-1102) while building hands-on lab experience to go along with it. Every lab I complete gets documented here: what I built, what I did, what I learned, and any problems I ran into and how I fixed them.

**Career path:** IT Help Desk → IAM Analyst → Cloud Security
