# OS Labs

Labs covering operating system skills for CompTIA A+ (Core 1 & Core 2).

## Topics Covered

- Windows 10/11 installation and configuration
- File system management (NTFS, permissions)
- Command line tools (cmd, PowerShell basics)
- Task Manager, Services, and MSConfig
- Registry editor basics
- Disk management and partitioning
- User account management
- Windows Update and driver management

## Tools Used

- **VirtualBox** — Windows 10 VM
- Windows Command Prompt and PowerShell
- Built-in Windows administrative tools
